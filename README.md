# Trello Clone Frontend

Interactive Kanban board interface for the Trello Enterprise task manager. 

## 🛠 Tech Stack
- HTML5 & CSS3
- Vanilla JavaScript (ES6+)
- HTML5 Drag and Drop API
- Fetch API for REST communication

- Dynamic column rendering (TODO, IN PROGRESS, DONE).
- Native Drag & Drop for task status updates.
